Join the tilde.town ~ring!

tilde.town ~users can join by pasting this html code in their index.html:


    <div id="tilde_ring" style="background-color:white; text-align:center; border-radius:10px;">
      <p><em>a member of the tilde.town ~ring</em></p>
      <p> <a id="tilde_town_ring" class="tilde_ring_link" href="">Random ~User</a>
        | <a id="random_tildebox" class="tilde_ring_link" href="">Random ~Box</a> </p>
      <p><a href="http://tilde.town/~um/tilde_ring/join.html">Join</a></p>
      <script type="text/javascript" src="http://tilde.town/~um/js/tilde_ring.js"></script>
    </div>

You can alter or edit the html in any way you choose, with the exception of four elements.

  • the random ~user link url and id
  • the random ~box link url and id
  • the join link url
  • the script tag must be included somehwere on your page
For an example of the flexibility of the ~ring parts, find ~selfsafem's implementation!

Please note: this is still in beta. Please email ~um@tilde.town or ~dan@tilde.town if there are any problems, or catch us in irc.
